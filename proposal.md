
# What I'm Building

- I am building an app that fetches live events from Ticketmaster’s Discovery API, lets users search by artist, venue, city, or keyword, and displays results.

### Which API I'm Using:

- I am using Ticketmaster and the url is https://developer.ticketmaster.com/products-and-docs/apis/getting-started/. 
Why I chose this: This data is interesting to me because I love going to concerts but there are a lot of websites that sells tickets which gets confusing. I would want a personal app that has all of the events and is trustworthy. While shopping for concert tickets I noticed a lot of websites can scam you or crash in the middle of buying tickets. 

## Core Features

- live event search
- real time filtering
- event detailed pages
- saved favorites
